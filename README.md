# Portfolio Game Development with Vue 3 + Vite 🎮✨

Welcome to my **Portfolio**! This project showcases my journey and achievements in **game development** using **Vue 3** and **Vite**. It demonstrates my skills in crafting interactive, visually stunning, and efficient web-based applications. 🚀

## About the Portfolio 📂

This portfolio serves as a window into my work, highlighting:
- Game projects I've developed 🕹️
- My coding expertise with **Vue 3** and modern web technologies 🌐
- An interactive and responsive design built for seamless navigation across devices 📱💻

## Key Features 🌟
- **Dynamic Components**: Leveraging Vue 3 `<script setup>` for clean and modular code.
- **Efficient Development**: Powered by **Vite** for blazing-fast hot module replacement (HMR) and build speed.
- **Interactive UI**: Engaging animations and transitions to enhance user experience.
- **Game Previews**: Showcase of gameplay and behind-the-scenes mechanics.
